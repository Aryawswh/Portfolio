Project Information

  • Project Tittle: Design of Control System and Safety System in Production Separator Based on PLC and HMI
  • Field: Process Control & Industrial Automation
  • Tools Used: EcoStruxure Machine Expert, Vijeo Designer, AutoCad

Project Description

The main focus is the automatic control of the fluid separation process, as well as the prevention of hazardous conditions through interlocks and shutdown systems.

1. P&ID Analysis to SAFE Chart

The process starts with a P&ID (Piping and Instrumentation Diagram) analysis to understand the fluid path, location of instruments (such as pressure, level, temperature transmitters), and actuator equipment (valves). From the P&ID, critical parameters for control and safety systems are determined. Next, a SAFE Chart is prepared in AutoCAD, which describes the relationship between:

  • Process Variable (for example: separator pressure is too high)
  • Alarm logic (warning to operator)
  • Shutdown logic (for example: closing inlet valve in case of overflow)
2. Control Logic (PLC Logic)
  • System built using EcoStruxure Machine Expert, with ladder logic
  • Analog signals of 4-20 mA are simulated and scaled into process variables, such as level (%) or pressure (psig), using linear formulas
  • Create alarm and shutdown logic, such as: when the level exceeds the high-high limit, the system will enter the shutdown state, the valve will close automatically
  • Operation is available in two modes: Manual and Auto
3. HMI design
  • HMI pages include Overview, Alarm, and Interlock Status
  • Colors and animations are used to clarify status, such as red, yellow, green
  • HMI features manual control for the Level Control Valve (LCV), allowing operators to open or close the valve directly
  • The system also features a graphical display of LCV opening trends against time, making it easy to monitor control performance in real-time
4. Automatic Control System
  • The system controls the Level Control Valve (LCV) automatically based on the input from the Level Transmitter
  • The purpose is to maintain the optimal level in the separator and prevent overflow or liquid void
  • Supports process stability and operational safety
5. Simulation & Validation
  • Systems are validated through logic and hmi simulations to ensure alarm and shutdown scenarios run as designed
  • Tests cover normal, abnormal, and trip logic conditions thoroughly
