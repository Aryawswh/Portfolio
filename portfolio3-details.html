Project Information

  • Project Tittle: Design and Implementation of a Smart Medicine Box Based on IoT and Android
  • Field: Internet of Things (IoT), Mobile Application Development
  • Tools Used: Arduino Nano, ESP8266(NodeMCU), Arduino-IDE, Android Studio, Firebase Realtime Database

Project Description

This project is the implementation of an IoT and Android-based Smart Medicine Box designed to help patients manage their medication intake automatically and on time. The system not only provides reminders through voice alerts but also automatically opens and closes the medicine box and monitors patients' medication intake.

The system uses Arduino Nano as the main controller, supported by several sensors and actuators such as the DS1302 RTC module for time setting, the MLX90614 temperature sensor for measuring the patient's body temperature without contact, and a servo motor for opening and closing the medicine box lid. In addition, there is a buzzer as an audible reminder and an HC-SR04 ultrasonic sensor to detect the presence of the patient's hand when taking medicine.

For connectivity, an ESP8266 (NodeMCU) connected to Firebase Realtime Database is used. Patients can set their medication schedule through an Android app connected to Firebase. When the time matches the set schedule, the buzzer will sound and the servo motor will open the medicine box. When the patient takes the medication, an ultrasonic sensor detects a change in distance as a sign that the medication has been taken. After the patient's hand is removed, the system automatically closes the medication box and turns off the buzzer.

Body temperature measured by the MLX90614 is also displayed on a 16x2 LCD screen via an I2C interface, allowing patients to monitor their body temperature directly on the device.

This project integrates various technologies such as embedded systems, wireless communication, cloud databases, and Android mobile app development, providing a smart and practical solution to improve medication adherence and monitor patient health, particularly for the elderly or patients with regular medication schedules.
